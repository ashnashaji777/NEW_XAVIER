PROGRAM OVERVIEW

The Master of Computer Applications (MCA) program is a postgraduate degree program designed to provide individuals with a strong foundation in computer science and applications. It is an in-depth course that combines theoretical knowledge with practical skills to prepare students for careers in the IT industry and related fields. Programming Languages and Algorithms: This area focuses on fundamental programming concepts and techniques. It covers programming languages such as C, C++, Java, and Python, and emphasizes algorithm design and analysis. Students learn to write efficient and optimized code and solve complex problems using algorithms.
Data Structures and Algorithms: This course delves deeper into the study of data structures, such as arrays, linked lists, stacks, queues, trees, and graphs. It also covers advanced algorithms for sorting, searching, and manipulating data. Understanding data structures and algorithms is crucial for efficient software development and problem-solving. Operating Systems focuses on the fundamental concepts and components of operating systems. It covers topics such as process management, memory management, file systems, and device management. Students gain a deep understanding of how operating systems work and how to develop efficient and secure software that interacts with them. Computer Networks explores the concepts and protocols of computer networks. It covers topics such as network architectures, network protocols, TCP/IP, routing, switching, and network security. Web Technologies focuses on the development of web-based applications and technologies. It covers topics such as HTML, CSS, JavaScript, server-side scripting, web frameworks, and web services. Software Development Tools and Techniques introduces students to various tools and techniques used in software development. It includes topics such as version control systems, integrated development environments (IDEs), software testing frameworks, and debugging techniques.

DURATION

2 Year fulltime programme

ELIGIBILITY

Aggregate 50% marks (45% for reserved category) in Graduation from any Recognized University. Candidates must appear any of the entrance exam like XAT/CAT/MAT/C-MAT/N-MAT/K-MAT/G-MAT/K-MAT/PGCET

SPECIALIZATIONS

St. Xavier’s Bangalore Offer’s Dual Specialization For MCA .

1 Web programming
2 Cloud Computing
3 Cryptography & Network Analysis
4 Big Data Analytics
5 Machine Learning

Note: Minimum 10 Students Required for each specialisation
